# 🇷🇺 Baza: Rus Tili

O'zbeklar uchun rus tili o'rganish dasturi — Telegram Mini App.

## Xususiyatlari

- **26 ta dars** — 210 ta haqiqiy so'z va ibora
- **3 bosqichli o'rganish**: O'rganish → Yozish → Sinov
- **Grammatika tushuntirishi** — har bir so'z uchun
- **Spaced Repetition** — so'zlar avtomatik takrorlanadi
- **Admin Panel** — darslar va foydalanuvchilarni boshqarish

## O'rnatish

```bash
npm install
npm run dev
```

## Deploy (Vercel)

1. GitHub ga push qiling
2. Vercel ga import qiling
3. Tugadi!
